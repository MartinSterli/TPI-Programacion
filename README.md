## Descripcion del proyecto

Este proyecto es una aplicacion de consola desarrollada en Python que permite gestionar informacion sobre paises del mundo. El programa lee un archivo CSV con datos de paises y permite agregar, actualizar, buscar, filtrar, ordenar y ver estadisticas de los mismos.

## Dataset utilizado

Archivo: paises.csv
Contiene datos de 10 paises con las siguientes columnas: nombre, poblacion, superficie, continente.

## Estructura del proyecto

- main.py : codigo fuente principal del programa
- paises.csv : dataset base con los paises cargados
- README.md : documentacion del proyecto

## Instrucciones para ejecutar el programa

Asegurarse de tener Python 3 instalado. Desde la carpeta del proyecto ejecutar:

    python main.py

El archivo paises.csv debe estar en la misma carpeta que main.py.

## Funcionalidades

- Agregar un nuevo pais
- Actualizar poblacion y superficie de un pais existente
- Buscar pais por nombre, admite coincidencia parcial
- Filtrar paises por continente, rango de poblacion o rango de superficie
- Ordenar paises por nombre, poblacion o superficie de forma ascendente o descendente
- Ver estadisticas generales: pais con mayor y menor poblacion, promedios y cantidad de paises por continente
- Mostrar todos los paises cargados
